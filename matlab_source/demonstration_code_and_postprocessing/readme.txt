
This code will automatically search for experiments where the data is saved in a folder that has the string 'exp'.
Therefore, you should rename the folders for example

from
/interaction_data_plate_on_table/151029_185257_good
to
/interaction_data_plate_on_table/exp151029_185257_good

The same main file will be used for two types of task:
1. Grasp a plate on a shelf.
2. Grasp a plate on a table.
